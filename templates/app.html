    🔍 What This Scanner Does

    This tool performs security scans on public API endpoints to detect common vulnerabilities like SQL Injection, Cross-Site Scripting (XSS), and CSRF. It sends crafted payloads to test input validation, responses, and security headers. Upon completion, it generates a detailed report for developers and security professionals.
